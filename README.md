# Проект: реальный кейс компании «СКАН»

### Установка:

1. Установить зависимости с помощью комманды npm install.
2. Запустить сервер с помощью комманды npm run dev.

### В проекте использованы:

- Vite.
- React Router 7.5.
- Redux.
- React Context.
- React Hook Form для работы с формами.
- Tailwindcss.
